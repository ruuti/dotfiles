# Usage

## Configure git

Create `~/.gitconfig-local` with the following content:

```
[user]
  name = xxx
  email = xxx
  signingkey = ~/.ssh/...
...
```

## Install

```
make brew-install 
```

```
./setup
```

```
source ~/.zshrc && source ~/.zshenv
```

```
make scripts-install 
```

Install nvm: https://github.com/nvm-sh/nvm

## Modifier Keys

- Caps Lock -> Escape
