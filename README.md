# Usage

## Configure git

Create `~/.gitconfig-local` with the following content:

```
[user]
  name = xxx
  email = xxx
  signingkey = xxx
...
```

## Install

```
make brew-install 
```

```
./setup
```

```
source ~/.zshrc && source ~/.zshenv
```

```
make scripts-install 
```

## Modifier Keys

- Caps Lock -> Escape
